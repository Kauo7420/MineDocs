术语入门

术语俚语信息

本篇指南仍在持续完善中，如有错误缺漏或待改进处，请及时反馈，谢谢！

近些年来，Minecraft社区在不断壮大，社区中自然而然地形成了一些独特的行话和术语。它们可能来源于游戏本身的特点、玩家间的互动习惯或者是某些流行事件的影响。了解这些专有名词不仅能让你更快地融入社区，还能帮助你更好地理解其他玩家的对话，从而提高游戏体验。

主要内容

本篇指南将涵盖一系列非正式术语及其含义，包括但不限于游戏内的特定操作、物品的昵称以及玩家间常用的俚语。我们会尽量用通俗易懂的语言来解释这些术语，让你即使没有深入研究也能轻松掌握它们的意义。

列表

服务器类型术语

名称寓意
养老服顾名思义，就是一个以自由自在生活、休闲为主要玩法的服务器，比如种田、建家、圈地、畜牧、钓鱼一类的玩法。
群组服/BC服一种服务器类型，利用群组服核心将不同的子服联合成一个整体服务器的服，具体可以看文档中的服务端入门。
BC服则是指用BungeeCord服务器核心架设的群组服，整体意思大致跟群组服的意思一样。
生电服生，及生存模式，电，即红石电路，生电的意思就是生存模式下的红石电路机械玩法，这类服务器主要以生存+红石为主要玩法，是生电圈的一种组织。
无政府服务器（又称无规则服务器）就是没有规则、没有正规管理（不是完全不管）、没有限制的三无服务器，玩家可以通过不正当手段进行游戏内竞争。这种服务器一般很混乱，而且不是一般服主能开的服务器。
无政府服服主只办三件事：配置，配置，还是他*的配置。
HVH服务器HVH，即Hacker VS Hacker，中文名为外挂对外挂，即外挂对战外挂的服务器。开设这种服务器的人一般就图一乐，然后比拼谁的挂更强罢了。
骗赞服字面意思，利用观众老爷的点赞量来控制服务器玩家的数量，比如1w点赞拉一个人，以这种手段骗取观众的点赞，使其达到博取眼球的效果。这种服务器……怎么说呢，一般质量比较差。
圈钱服
快餐服		字面意思，服主在网上随便找一个服务器整合包（模板），然后修改一下服务器信息，当自己原创的服务器开设起来，然后往里面塞入氪金系统，然后大肆宣传，骗取玩家的小钱钱，然后拿钱走人（跑路）。总结就是一句话：模板开服，服内圈钱，卷钱跑路。
神犬服一种说法是网易版联机房间，该作风是网易版MC的经典，也是常见现象。比如：我是公主，你们都必须给我干活 =)
另一种说法是指某些私服管理员利用个人权限“公报私仇”，对普通玩家实行封禁、禁言等一系列非正常管理措施。

技术术语

名称寓意
压测全称压力测试，是指利用某些手段（比如塞入一堆假玩家）使服务器的资源占用急速上升，以达到测试服务器性能极限的效果。但是如果利用不当，则会成为熊孩子攻击服务器的一种手段，让服务器瘫痪一阵子，严重的话就可以导致服务器损坏。
后门一种网络病毒，指绕过安全控制而获取对程序或系统访问权的方法。后门的最主要目的就是方便以后再次秘密进入或者控制系统。这种病毒一般来源于不正规渠道获取的软件等等，比如在不知名的网站上下载的插件，装到服务器上后可让你的服务器遭受损失。
注入就是把一些功能注入到客户端内，以此达到篡改客户端行为的目的，一般代指外挂注入，是挂B的基操。
分支开发者术语，一种项目的一个子版本就是分支，详细一点说就是指从主项目派生出的新版本或代码线，用于开发新功能或定制用途。在软件开发中，分支有助于隔离更改，便于测试和集成。
抽象化理解： A项目是率先开发的，是原创，随后又蹦出来个B项目，不过B项目是基于A项目开发的，用的都是A项目的东西，进行二次创作，创作出一个全新的版本，这就是分支。
DockerDocker 是一个开源的容器化平台，允许开发者将应用程序及其所有依赖项打包成一个轻量级、可移植的容器。这些容器可以在任何支持 Docker 的环境中运行，包括本地开发机器、测试服务器和生产环境。Docker 使用 Linux 容器技术，如 cgroup 和 namespace，来实现进程隔离，使得每个容器都有独立的文件系统、网络和进程空间。
Docker镜像(Docker Image)一个 Docker 镜像包含运行容器化应用程序所需的一切。 （例如用于 Minecraft 服务器的 Java）。
容器(Container)每台服务器都将在一个隔离的容器内运行，以强制执行硬件限制（例如 CPU 和 RAM）并避免一个节点上的服务器之间产生任何干扰。这些都将由 Docker 创建。
DDoSDDoS（Distributed denial of service attack），中文名为分布式拒绝服务攻击，DDoS攻击以网站和服务器为目标，攻击方式是中断网络服务。这种攻击可以让服务器持续一段时间处于宕机、崩溃状态，让服务器无法正常运行。如果可能，攻击者也可以趁机窃取一些用户隐私数据。
DHCPDHCP（Dynamic Host Configuration Protocol，动态主机配置协议）是一种网络管理协议，它允许服务器自动分配IP地址和其他网络配置参数给网络中的客户端设备，如计算机、智能手机、打印机等。通过使用DHCP，网络管理员可以简化网络配置的过程，减少手动配置IP地址的工作量，并提高网络的可管理性和灵活性。
DNSDNS（域名系统）对于Minecraft服务器服主来说，是一个重要的工具。它允许你将服务器的IP地址与一个易于记忆的域名关联起来，这样玩家就可以通过输入域名来连接到你的服务器，而不用记住复杂的IP地址。例如，你可以将你的服务器设置为“play.server.com”，玩家只需在Minecraft客户端中输入这个域名，就能连接到你的服务器。通过设置DNS记录，如A记录或CNAME记录，你可以将域名指向服务器的IP地址。此外，使用SRV记录还可以隐藏服务器的端口号，使玩家只需输入域名即可连接，而无需输入端口。这样不仅方便了玩家，也让你的服务器看起来更加专业。
抽象化理解：114.514.191.981(IP地址) -> play.server.com(通过A协议的DNS解析后的地址地址)
网络传输协议
TCP/UDP		TCP和UDP都是一种网络传输协议。网络传输协议是计算机网络中不同设备进行数据交换时必须遵守的一系列规则和标准。这些协议定义了数据的传输格式、速率、步骤以及如何处理可能出现的错误。网络传输协议可以在硬件、软件或两者之间实现，确保信息能够在传输系统中的终端之间顺利传播。
TCP是一种面向连接的协议，强调数据传输的可靠性和顺序性。它通过建立连接、数据确认和错误检查等机制，确保数据包按顺序、无错误地到达目的地。如果数据包丢失或出错，TCP会自动重传，直到数据完整传输。这种可靠性使得TCP适用于需要准确数据传输的应用，如文件传输、电子邮件和网页浏览。在Minecraft中，TCP主要用于Java版服务器中，处理主要的游戏数据传输，如玩家动作、方块变化等。
UDP则是一种无连接的协议，优先考虑传输速度而非可靠性。它不建立连接，也不保证数据包的顺序或完整性。UDP适用于对实时性要求高的应用，如在线游戏、视频流和语音通话，这些场景中少量的数据丢失对用户体验影响不大。在Minecraft中，UDP用于基岩版服务器中，处理一些次要的数据传输，如心跳包和聊天信息。
子网掩码子网掩码是一种用于区分IP地址中网络部分和主机部分的工具，它和IP地址一样，通常以点分十进制表示，例如255.255.255.0。在服务器的网络配置中，子网掩码帮助确定哪些设备属于同一子网，这对于局域网内的联机游戏非常重要。例如，如果你的子网掩码是255.255.255.0，那么你的子网可以容纳256个IP地址，这意味着你可以将多台设备配置在同一个子网内，以便它们能够相互通信和连接到你的服务器。
MAC地址MAC地址（Media Access Control address）是网络设备（如电脑、路由器、交换机等）网卡的唯一硬件标识符，它由12个十六进制数字组成，通常以冒号或破折号分隔，例如00:1A:2B:3C:4D:5E。在服务器的网络配置中，MAC地址用于在网络层以下唯一标识设备，确保数据包能够准确地发送到正确的硬件设备。虽然在日常的服务器管理中，你可能不会直接与MAC地址打交道，但在某些情况下，比如网络故障排查或设置网络访问控制时，了解设备的MAC地址可能会很有帮助。
带宽带宽是指网络连接的传输能力，即单位时间内可以传输的数据量，通常以比特每秒（bps）为单位来衡量。带宽决定了服务器与玩家设备之间数据传输的速度，包括玩家的指令、游戏状态更新、方块变化等信息的传输。如果带宽不足，可能会导致玩家在游戏过程中出现延迟、卡顿甚至掉线的情况，影响游戏体验。因此，确保服务器有足够的带宽是提供流畅游戏体验的关键因素之一。
防火墙防火墙是一种网络安全系统，它监控和控制进出服务器的网络流量，基于预定义的安全规则来决定哪些流量是允许的，哪些是被阻止的。防火墙的主要功能包括过滤网络流量、保护服务器免受网络攻击、恶意软件和未经授权的访问。在Minecraft服务器的上下文中，防火墙可以防止恶意流量和攻击，确保只有合法的玩家连接能够进入服务器。例如，你可以配置防火墙规则，只允许特定的端口（如Minecraft默认的25565端口）的流量通过，从而减少潜在的安全风险。
QoSQoS（Quality of Service，服务质量）是一种网络管理技术，用于优化网络资源的分配，确保关键应用（如你的MC服务器）的网络流量优先得到处理。通过配置QoS，你可以优先保障Minecraft服务器的网络带宽，确保玩家的游戏体验更加流畅，减少延迟和卡顿。例如，在网络拥塞时，QoS可以确保Minecraft服务器的数据包优先传输，而不会被其他非关键流量（如文件下载或视频流）抢占带宽。
VPNVPN（虚拟专用网络）是一种通过加密协议和隧道技术将用户的通信数据“隧道”起来的网络技术，它允许用户通过公共网络建立一个安全的、专用的连接。使用VPN可以访问被封锁的MC服务器，保护玩家的隐私和数据安全，避免流量监控和审查。例如，如果某些MC服务器在国内被封锁，玩家可以通过连接到国外的VPN服务器来访问这些服务器。
VLANVLAN（虚拟局域网）是一种网络技术，它允许你将一个物理局域网（LAN）在逻辑上划分为多个独立的广播域。每个VLAN都是一个独立的广播域，VLAN内的设备可以直接通信，而不同VLAN之间的设备则不能直接通信，除非通过路由器或三层交换机进行路由。这种划分方式可以有效减少广播流量，提高网络性能和安全性。例如，你可以将服务器的管理设备和玩家设备划分到不同的VLAN中，从而隔离管理流量和游戏流量，减少广播风暴的影响，同时增强网络的安全性。

通用术语

名称寓意
神犬原名神权，神犬是贬义代词。这个词的意思是指利用个人的权力肆意妄为，大肆搞反动型破坏，想方设法利用手里的权力实现自己的“皇帝梦”。有这种行为的人，Ta的素质一般很差。
圈钱一般意思就是利用某种手段骗取玩家的钱，让服主赚的满满当当，实现自己的发￥梦。
跑路字面意思，就是丢下原有的一切（如服务器）不管，带着跟自己利益有关的东西（比如赚的💰）或在破坏掉现有的一切之后逃跑到别的地方（反正是你看不到的地方），消失得无影无踪。
灵魂出窍一种辅助型外挂，让玩家（的视角）脱离限制，自由飞到任意地方来查看各个地方，相当于透视挂。虽然不会对其他方块产生影响，但是也有可能让玩家触及到自身不能碰到的地方，比如墙后。
补票“补票”是指玩家在预先体验过付费游戏后再去购买正版游戏的行为。
也指玩家之前一直在玩无需付费的“破解版”，并没有购买该游戏，之后再去购买游戏的行为。
黑卡部分正版玩家可能由于资金不充足或没有足够的购买能力，往往选择在第三方平台购买价格相对低廉的正版账号，并且通过卖家更改账号的信息以获取该正版账号。而我们将以这种方式获得的正版账号称为“黑卡”。注意，它并不能保证能够为玩家的正常游戏永久地保驾护航，也有可能会被封号。

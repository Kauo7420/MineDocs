在Minecraft服务器搭建与管理的过程中，常常需要参考官方文档、下载插件、模组或获取帮助。本附录整理了常用的相关网站，方便小白快速查找所需资源。

1. 官方资源

网站说明链接
Minecraft 官网官方游戏购买、新闻及更新信息https://www.minecraft.net
Mojang 帐号管理管理 Mojang / Minecraft 帐号https://account.mojang.com
Minecraft Wiki官方维基百科，包含游戏机制、方块、命令等https://minecraft.fandom.com
官方服务器软件（Vanilla）下载原版服务器程序https://www.minecraft.net/en-us/download/server

网站说明链接

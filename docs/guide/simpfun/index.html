简幻欢是什么？能吃吗？

「简幻欢」是一个「服务器联机平台」，英文名为SimpFun，孵化自简幻云团队 ，是团队下最新的公益项目，目前已创建数以十万级的服务器，已是数万用户进行游戏服务器联机的选择，

简幻欢诞生伊始，只是希望能够给广大玩家提供一个好用的服务器远程联机平台，达成「人人都可免费开属于自己的服务器」这个小目标。所以，带着这颗初心，我们开启了简幻欢项目，致力于为每个想开开服的提供便利的服务，像自来水的水龙头一样即开即用，让所有人可以在「简幻欢」中平等快乐地开放自己的服务器，让再小的团体也可以自由进行游戏联机。

简而言之就是，简幻欢是一个公益免费的服务器托管平台，也是一个免费可用的面板服。

我为什么要用你？用云服务器不好吗？

因为简幻欢目前是国内为数不多的公益性质的免费服务器平台，支持多款游戏的服务器联机，与传统内网穿透联机相比有无需本地挂机，延迟更低等优势。

优势

简幻欢有以下明显优势：

高性能CPU

高频高性能处理器，提供卓越算力支持

三线BGP网络

高速互联网接入，单实例1Gbps超大带宽

企业级存储阵列

RAID10固态阵列+企业级机械阵列双重保障

简单易上手的UI操作

采用SFE4.0面板操作UI，隔壁60岁的老奶奶都能学会操作
